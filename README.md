# Portfolio Rebalance

Static portfolio rebalance page for GitHub Pages.

## GitHub Pages

Upload these files to a GitHub repository:

- `index.html`
- `styles.css`
- `app.js`
- `.nojekyll`

Then enable GitHub Pages from the repository settings.

`portfolio-rebalance.html` is the single-file version for local sharing.

`server.mjs` and `start-site.cmd` are only for local live-price proxy use.
